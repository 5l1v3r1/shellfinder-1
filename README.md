
# shellfinder

A Simple Tool to Find Shells and Some Interesting Endpoints in Websites

## Installation 
Prerequisites:

- Python 3.6

- requests module

1. Put your websites in `websites.txt`

2. Install requests `pip3 install requests`

3. Run the script `$ python3 shellfinder.py`
